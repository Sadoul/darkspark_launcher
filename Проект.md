# DarkSpark Launcher

Лаунчер для игры с тёмной темой (rgb 23,49,48 и rgb 189,22,51).

## Что сделано:
1. Ребрендинг: RPW → DarkSpark
2. Тёмная тема по умолчанию
3. Вход только по нику (офлайн)
4. Иконки обновлены
